6:栈方法（后进先出）

  • push()：在数组的末尾增添项，并返回修改后的数组的长度；
  • pop():从数组中末尾中移走一项，减少length的值，返回移除的项；

7：队列方法（先进先出）

  • shift():移除数组中的第一项并返回该项，同时将数组的长度减一；

  • unshift():在数组前端加上任意项并返回新数组的长度；

  • push()和shift()组合可以使用队列一样使用数组；

  • pop（）和unshift（）组合可以使用队列一样使用数组；

8：重排序方法

  • reverse()：反转数组项的位置；

  • sort()：按升序排列数组项；

9：操作方法

  • concat():基于当前数组的所有项创建一个新的数组；
  • slice():基于当前数组一个或多个项创建一个新数组；
  • 删除，插入，替换

10：位置方法

  • indexOf():从数组的开头开始向后查找；
  • lastIndexOf():从数组的末尾开始向前查找；

11：迭代方法

  • every():每一项都符合给定的函数，最后返回ture，否则，返回false；
  • some（）：至少一项符合给定的函数，最后返回ture，否则，返回false；
  • filter（）：运行给定的函数后，返回ture的项组成的数组；
  • forEach（）：运行给定的函数，但是没有返回值；
  • map（）：运行给定的数组，返回运行后的值组成的数组；

12：归并方法
reduce（）：从数组的第一项开始，遍历整个数组；

var values=[1,2,3,4,5];
var sum=values.reduce(function(prev,cur,index,array){
         return prev+cur;
         });
         alert(sum);//15

reduceRight（）：从数组的最后一项开始，遍历整个数组；

     var values=[1,2,3,4,5];
var sum=values.reduceRight(function(prev,cur,index,array){
         return prev+cur;
         });
         alert(sum);//15
